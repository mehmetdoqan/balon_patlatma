🎈 Balon Patlatma
https://mehmetdoqan.github.io/balon_patlatma/

Çocuklar için tarayıcı tabanlı, blok kodlama mantığıyla çalışan eğitici bir bulmaca oyunu.
Hiçbir kurulum gerektirmez — tek bir HTML dosyasıdır, tarayıcıda açılır.

🎮 Oyun Hakkında

İğneyi yönlendirerek tüm balonları patlat! Komutları sürükle-bırak veya tıklayarak sıraya koy, ardından BAŞLAT butonuna bas.

Oyun, çocuklara sıralı düşünme, algoritma kurma ve hata ayıklama becerilerini eğlenceli bir şekilde kazandırmayı hedefler.

🕹️ Nasıl Oynanır?
Zorluk seviyesi seç: 🟢 Kolay / 🟡 Orta / 🔴 Zor
Haritadan bir bölüm seç
Yön butonlarına tıkla veya sürükle-bırak ile komut kuyruğunu oluştur
📌 Patlat komutuyla balonun üzerindeyken patlatma işlemi yap
🚀 BAŞLAT butonuna bas
İğne komutları sırayla uygular
Tüm balonları patlatırsan bölümü geçersin!

💡 Komut çerçevesinin dışına sürüklersen komutu silebilirsin.
💡 Çerçeve içindeki komuta tıklayarak da silebilirsin.

📋 Komutlar
Buton	Açıklama
⬆️	Yukarı git
⬇️	Aşağı git
⬅️	Sola git
➡️	Sağa git
📌	Balonu patlat
🗺️ Bölümler
60 bölüm — 20 Kolay + 20 Orta + 20 Zor
Her bölümde farklı balon konumları ve duvar engelleri
Tamamlanan bölümler haritada ⭐ ile işaretlenir
Bölümler sırayla açılır
🧱 Harita Öğeleri
Sembol	Açıklama
📌	İğne (oyuncu)
🎈	Balon (hedef — üstüne gel ve patlat!)
🧱	Duvar (engel — üstüne girilirse hata)
🔊 Ses Efektleri
Hareket sesi — her adımda
Patlama sesi — balon patlatıldığında (yüksek frekans → hızlı düşüş)
Başarı melodisi — bölüm tamamlandığında
Duvar sesi — engele çarpıldığında
Üzgün melodi — balonlar patlatılamadığında
✨ Animasyonlar
Balonlar hafifçe yukarı-aşağı sallanır (float animasyonu)
Patlama anında büyüyüp küçülme + 8 yönde parçacık efekti
Hata durumunda ekran sarsılması
🛠️ Editör Modu (Öğretmenler İçin)

Oyun ekranındaki ⚙️ butonuna tıklayarak harita editörüne erişebilirsin.

🔐 Editör şifre korumalıdır — yalnızca öğretmenler içindir.

Editörde yapabileceklerin:

📌 İğnenin başlangıç konumunu değiştir
🎈 Balon ekle / kaldır
🧱 Duvar ekle / kaldır
